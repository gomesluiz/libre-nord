Two Distinct Atmospheres

Nord Dark

The classic dark theme, focused on deep, restful tones. Ideal for low-light environments, featuring high-contrast text and vibrant Aurora accents.

Nord Light

The bright variant. Designed for highly illuminated workspaces, it maintains maximum visual sharpness using deep blues and high-contrast typography.

How to Install

  1. Download the .oxt file of your preferred theme.
  2. Open LibreOffice and navigate to Tools > Extension Manager...
  3. Click Add, select the downloaded file, and confirm.
  4. Restart LibreOffice.
  5. Go to Tools > Options > LibreOffice > Application Colors.
  6. Select the theme in the Scheme dropdown and apply.
