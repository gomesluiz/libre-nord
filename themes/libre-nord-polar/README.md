# Nord Polar Night Theme

[⬅ Back to main repository](../../README.md)

**Polar Night** is the classic implementation of the Nord palette. It consists of a restful dark theme focused on the four night tones of the original palette (`nord0` to `nord3`). It was designed for low-light environments, offering an immersive reading and writing experience without straining the eyes.
