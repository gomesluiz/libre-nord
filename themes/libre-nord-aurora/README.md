# Nord Aurora Theme

[⬅ Back to main repository](../../README.md)

**Aurora** shares its dark foundation with the Polar Night theme but replaces the subtle highlights with vibrant and saturated colors (`nord11` to `nord15`). This extension was designed for users who need an immediate and aggressive differentiation between visual interface elements, providing touches of purple, green, and yellow over a night background.
